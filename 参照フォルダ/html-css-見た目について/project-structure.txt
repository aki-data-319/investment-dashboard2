投資ダッシュボード - プロジェクト構造

## メインファイル構成
app/
├── page.tsx                    # メインダッシュボード画面
├── layout.tsx                  # レイアウト設定
├── globals.css                 # グローバルスタイル（カラーパレット含む）
└── holdings/
    └── page.tsx               # 保有銘柄管理画面（エクセル形式）

components/
├── monthly-chart.tsx          # 月次パフォーマンスチャート（期間選択タブ付き）
├── sector-chart.tsx           # セクター配分チャート（種類選択タブ付き）
├── investment-modal.tsx       # 銘柄追加・編集モーダル
└── delete-confirm-dialog.tsx  # 削除確認ダイアログ

data/
└── test-data.ts              # テストデータファイル

## 主要機能
1. メインダッシュボード
   - 月次パフォーマンス（投資額エリアグラフ、期間選択：3ヶ月/6ヶ月/1年/2年）
   - セクター配分（円グラフ、種類選択：地域別/業種別/資産別）
   - 資産サマリー（総投資額、月額積立、評価損益）

2. 保有銘柄管理（エクセル形式）
   - 全列表示テーブル（銘柄名、ティッカー、種類、保有数、価格、評価額、損益等）
   - 投資信託の構成銘柄展開表示
   - 検索・フィルター機能（銘柄名、セクター、地域）
   - ソート機能（銘柄名、評価額、リターン、セクター順）
   - 列表示・非表示選択機能
   - セクター配分自動計算（構成銘柄含む）
